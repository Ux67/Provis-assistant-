# Provis-assistant-
Ai assistant with basic on screen control ,wikipedia support and calculator 
it has features such as :- 

# Voice commmand 
  > provis assistant uses pyttsx3 for voice translation.
  > this helps to build a good user interface.

# Speech recognition:-
  > speech recognition module helps to understand user commands to execute it.

# Time and DateTime module:-
  > this module helps to show current date and time.

# Os module:- 
  > this module helps to open operating system apps.
  > for ex:- youtube, chrome , system apk ,etc.

# Webbrowser module:- 
  > this module helps to search a specific topic from the web to display it.

# numpy module:- 
  > this module helps the user solve a mathematical query.

# matplotlib tkinter module:-
  > for ai voice behavioural pattern representation and voice command button to stop or start the conversation.

Note:- 
  install all the required module by using pip install (module) to remove module errors.

Meaning of PROVIS:-
python recreated online virtual system
